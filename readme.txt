CodesWholesale.com integration plugin for WooCommerce.

== Description ==

This plugin allows to integrate a WooCommerce store with [CodesWholesale.com](http://codeswholesale.com) API allowing to make purchases on the platform automatic, also providing price and stock updates for the products.

== Installation ==

1. Upload and install the plugin through 'Plugins' tab.
2. Use the 'CodesWholesale' menu to configure your plugin

== Frequently Asked Questions ==

= What it does? =

1.  syncing your product listing with CodesWholesale
2.  buying and sending keys when the order is marked as complete
3.  error reporting and email notifications sent to the admin
4.  low balance notifications
5.  automatic order completion upon payment
6.  game assets import
7.  tracking past orders
8.  new filter options and CSV download
9.  rating the risk of incoming orders
10. invoice pdf download
11. brand-new notifications via postback URL
12. double-check price option
13. pre-order option
14. charm pricing at work

= How to generate client ID and Secret? =

1. Log in to [our platform](http://codeswholesale.com)
2. Go to *Web API* tab
3. Click on *Generate client credentials*
4. Please note the credentials somewhere - especially client secret as it will get pernamently hidden after you leave the page.

= What to do if I lost my client secret? =
1. Log in to [our platform](http://codeswholesale.com)
2. Go to *Web API* tab
3. Click on *Delete client*
4. Click on *Yes, I'm sure - delete it* (don't worry, it will not delete your account)
5. Click on *Generate client credentials*
6. You will receive a new set of client ID and secret - please note it somewhere and update your API settings
